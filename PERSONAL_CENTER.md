# 个人中心功能说明

## 功能概述

新增的个人中心功能为用户提供了完整的个人信息和资产管理界面，包括个人信息查看、资产概览、交易统计和资金流水等功能。

## 主要功能

### 1. 个人信息标签页
- **基本信息**：用户名、邮箱、电话、真实姓名等
- **身份信息**：身份证号、地址等
- **账户状态**：注册时间、最后登录时间、账户状态

### 2. 资产信息标签页
- **资产概览**：
  - 总资产：账户总价值
  - 可用资金：可用于交易的资金
  - 冻结资金：被冻结的资金
  - 持仓市值：当前持仓的市值
  - 浮动盈亏：实时盈亏情况
  - 盈亏比例：盈亏百分比

- **资产明细表格**：
  - 各项资产的详细金额
  - 占总资产的比例
  - 详细说明

### 3. 交易统计标签页
- **胜率显示**：交易成功率
- **统计信息**：
  - 总交易次数
  - 买入/卖出次数
  - 总交易量和交易金额
  - 总盈亏情况
  - 最后交易时间

### 4. 资金流水标签页
- **流水记录**：显示最近的资金变动记录
- **记录信息**：
  - 时间：操作时间
  - 类型：充值、提现、买入、卖出、分红等
  - 金额：变动金额（正数为收入，负数为支出）
  - 余额：操作后余额
  - 描述：操作说明
  - 状态：操作状态

## 界面特点

### 1. 数据可视化
- **进度条**：盈亏比例和胜率用进度条直观显示
- **颜色编码**：
  - 红色：盈利、收入
  - 绿色：亏损、支出
  - 不同颜色帮助快速识别数据性质

### 2. 实时更新
- 数据支持手动刷新
- 登录后自动加载用户数据
- 持仓变化时自动更新资产信息

### 3. 用户友好
- 清晰的标签页分类
- 直观的数据展示
- 统一的货币格式（¥符号）
- 百分比格式显示

## 使用方法

### 1. 查看个人信息
1. 登录后切换到"个人中心"标签页
2. 点击"个人信息"子标签
3. 查看个人基本信息和账户状态

### 2. 查看资产情况
1. 在"个人中心"标签页点击"资产信息"子标签
2. 查看资产概览和详细明细
3. 点击"刷新数据"按钮更新最新信息

### 3. 查看交易统计
1. 点击"交易统计"子标签
2. 查看交易次数、胜率等统计信息
3. 了解个人交易表现

### 4. 查看资金流水
1. 点击"资金流水"子标签
2. 查看最近的资金变动记录
3. 了解资金流向和余额变化

## 技术实现

### 数据模型
- `UserInfo`：用户基本信息
- `AccountInfo`：账户资产信息
- `CashFlowRecord`：资金流水记录
- `TradingStats`：交易统计信息

### 核心组件
- `UserService`：用户数据管理服务
- `UserInfoWidget`：个人中心界面组件

### 数据流程
1. 用户登录成功后，主窗口设置当前用户
2. UserInfoWidget加载用户相关数据
3. UserService生成模拟数据（实际应用中连接真实数据源）
4. 界面实时更新显示数据

## 模拟数据说明

当前版本使用模拟数据，包括：
- 随机生成的用户信息
- 模拟的资产数据
- 虚拟的交易统计
- 模拟的资金流水记录

## 扩展功能

### 可扩展的功能
1. **数据编辑**：允许用户修改个人信息
2. **图表展示**：资产变化趋势图
3. **导出功能**：导出交易记录和资金流水
4. **通知提醒**：重要信息推送
5. **安全设置**：密码修改、安全验证等

### 数据源集成
- 连接真实的用户数据库
- 集成实时资产计算
- 对接交易系统获取真实数据
- 连接银行系统获取资金流水

## 注意事项

1. 当前使用模拟数据，实际部署时需要连接真实数据源
2. 资产计算基于持仓和实时价格
3. 资金流水记录需要与实际交易系统同步
4. 个人信息涉及隐私，需要安全保护
5. 数据刷新频率可根据需要调整

## 安全考虑

1. 敏感信息（如身份证号）应进行脱敏处理
2. 资金信息需要加密存储和传输
3. 用户权限控制，防止越权访问
4. 操作日志记录，便于审计追踪
