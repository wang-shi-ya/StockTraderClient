# 股票交易客户端 - 新功能说明

## 新增功能概述

本次更新为股票交易客户端添加了以下主要功能：

### 1. 实时行情功能
- **实时价格显示**：显示股票的当前价格、涨跌幅、成交量等
- **买卖盘口**：显示五档买卖盘口信息
- **多股票监控**：可以同时监控多只股票
- **市场指数**：显示上证指数、深证成指等主要指数
- **板块行情**：显示各行业板块的涨跌情况

### 2. 历史数据功能
- **K线图**：支持多种周期的K线图（1分钟、5分钟、日K、周K等）
- **交互式图表**：支持鼠标悬停查看详细信息、缩放、平移
- **成交量图**：在K线图下方显示对应的成交量
- **历史成交明细**：查询和显示历史成交记录

### 3. 基本面信息
- **公司基本信息**：股票代码、公司名称、所属行业等
- **财务数据**：营业收入、净利润、总资产等财务指标
- **股本结构**：总股本、流通股本、市值等信息
- **公司公告**：显示相关公司公告信息

## 使用方法

### 实时行情
1. 点击"实时行情"标签页
2. 在"股票代码"输入框中输入股票代码（如：000001）
3. 点击"添加"按钮添加到监控列表
4. 可以查看实时价格、涨跌幅、成交量等信息
5. 切换到"市场指数"和"板块行情"标签查看大盘信息

### K线图
1. 点击"K线图"标签页
2. 选择股票代码和周期（1m、5m、15m、30m、1h、1d、1w）
3. 点击"刷新"按钮获取数据
4. 使用鼠标悬停查看详细信息
5. 使用鼠标滚轮进行缩放

### 公司信息
1. 点击"公司信息"标签页
2. 选择股票代码
3. 查看"基本信息"、"财务数据"、"公司公告"等标签页
4. 点击"刷新"按钮更新数据

## 技术实现

### 数据模型
- `QuoteData`：实时行情数据结构
- `KLineData`：K线数据结构
- `CompanyInfo`：公司信息数据结构
- `IndexData`：指数数据结构
- `SectorData`：板块数据结构

### 核心组件
- `DataService`：数据服务类，负责数据生成和管理
- `QuoteWidget`：实时行情显示组件
- `KLineWidget`：K线图绘制组件
- `CompanyWidget`：公司信息显示组件

### 模拟数据
当前版本使用模拟数据，包括：
- 随机生成的股票价格和成交量
- 模拟的K线数据
- 虚拟的公司财务信息
- 模拟的市场指数和板块数据

## 编译和运行

1. 确保已安装Qt 6.9.1和MySQL数据库
2. 打开Qt Creator
3. 打开`StockTraderClient.pro`项目文件
4. 配置项目并编译
5. 运行程序

## 注意事项

1. 当前版本使用模拟数据，实际使用时需要连接真实的股票数据源
2. K线图功能需要较大的显示区域，建议在较大屏幕上使用
3. 实时行情每秒钟更新一次，可根据需要调整更新频率
4. 数据库连接配置在`dbmanager.cpp`中，可根据实际环境修改
