# StockTraderClient 股票交易客户端

## 项目介绍

StockTraderClient 是一个基于 Qt 框架开发的股票交易客户端应用，提供实时股票行情查看、交易操作、个人信息管理等功能。

## 功能特性

### 核心功能
- **行情查看**：实时股票报价、K线图分析、公司信息查看
- **交易功能**：股票买卖、订单管理、持仓查看
- **个人中心**：用户信息管理、账户资产查看、交易历史记录
- **用户管理**：支持登录/注册、个人信息编辑（包括用户名修改）

### 新增功能
- **实时行情功能**：实时价格显示、买卖盘口、多股票监控、市场指数、板块行情
- **历史数据功能**：K线图（支持多种周期）、交互式图表、成交量图、历史成交明细
- **基本面信息**：公司基本信息、财务数据、股本结构、公司公告
- **个人中心功能**：个人信息标签页、资产信息标签页、交易统计标签页、资金流水标签页

## 技术栈

- **开发框架**：Qt 6.9.1
- **编程语言**：C++
- **数据库**：SQL
- **网络通信**：Qt Network

## 主要模块

- **Client**：客户端核心类，负责与服务器通信
- **DataService**：数据服务类，处理市场数据
- **UserService**：用户服务类，处理用户相关操作
- **MainWindow**：主窗口，集成各功能模块
- **QuoteWidget**：行情报价组件
- **KLineWidget**：K线图组件
- **CompanyWidget**：公司信息组件
- **UserInfoWidget**：用户信息组件
- **EditUserInfoDialog**：用户信息编辑对话框

## 安装与运行

### 环境要求

- Qt 6.9.1 或更高版本
- C++17 兼容的编译器
- 支持的操作系统：Windows、Linux、macOS

### 编译步骤

1. 打开 Qt Creator，选择 "打开项目"，导航到 `StockTraderClient.pro` 文件
2. 选择合适的构建配置（Debug 或 Release）
3. 点击 "构建" 按钮编译项目
4. 编译完成后，运行应用程序

### 直接运行

如果已有编译好的可执行文件，直接运行即可：
- Windows：`build/Desktop_Qt_6_9_1_MinGW_64_bit-Debug/debug/StockTraderClient.exe`
- Linux：`build/StockTraderClient`
- macOS：`build/StockTraderClient.app`

## 使用指南

### 登录/注册

1. 启动应用程序
2. 从菜单 "账户 → 登录/注册" 打开登录对话框
3. 输入用户名和密码登录，或点击 "注册" 按钮创建新账户

### 查看行情

1. 在主窗口的标签页中选择 "行情" 或 "K线图"
2. 在股票代码下拉框中选择或输入股票代码
3. 查看实时行情数据和K线图表

### 实时行情使用
1. 点击"实时行情"标签页
2. 在"股票代码"输入框中输入股票代码（如：000001）
3. 点击"添加"按钮添加到监控列表
4. 可以查看实时价格、涨跌幅、成交量等信息
5. 切换到"市场指数"和"板块行情"标签查看大盘信息

### K线图使用
1. 点击"K线图"标签页
2. 选择股票代码和周期（1m、5m、15m、30m、1h、1d、1w）
3. 点击"刷新"按钮获取数据
4. 使用鼠标悬停查看详细信息
5. 使用鼠标滚轮进行缩放

### 公司信息使用
1. 点击"公司信息"标签页
2. 选择股票代码
3. 查看"基本信息"、"财务数据"、"公司公告"等标签页
4. 点击"刷新"按钮更新数据

### 交易操作

1. 在主窗口的 "交易" 标签页中
2. 选择股票代码，输入交易价格和数量
3. 点击 "买入" 或 "卖出" 按钮提交订单
4. 在 "订单" 标签页中查看订单状态和历史记录

### 交易功能改进

#### 股票选择机制
- **改进**：只能从已添加到监控列表的股票中选择进行交易
- **界面变化**：将股票代码输入框改为下拉选择框，显示格式：`股票代码 (公司名称)`
- **自动更新**：当用户在"实时行情"标签页添加或移除股票时，交易页面的股票选择列表会自动更新

#### 智能功能
- **智能价格填充**：选择股票后，价格输入框会自动填充该股票的最新价格
- **预置股票**：程序启动时自动添加一些常用股票代码，如平安银行、万科A、浦发银行等
- **验证机制**：下单前检查是否选择了有效的股票代码

#### 使用方法
1. **添加股票到监控列表**：在"实时行情"标签页输入股票代码并点击"添加"按钮
2. **进行交易**：在"交易"标签页选择股票，输入数量和价格，点击"下单"按钮
3. **移除股票**：在"实时行情"标签页选择股票并点击"移除"按钮

### 个人中心使用

1. 在主窗口的 "个人中心" 标签页中查看个人信息
2. 点击 "编辑信息" 按钮修改个人信息，包括用户名
3. 查看账户资产和交易统计数据

### 个人中心详细功能
1. **个人信息标签页**：查看基本信息、身份信息、账户状态
2. **资产信息标签页**：查看总资产、可用资金、冻结资金、持仓市值、浮动盈亏等
3. **交易统计标签页**：查看胜率、总交易次数、买入/卖出次数、总交易量和交易金额、总盈亏情况
4. **资金流水标签页**：查看最近的资金变动记录，包括时间、类型、金额、余额、描述、状态

## 功能亮点

1. **实时行情**：提供实时股票报价和K线图分析
2. **便捷交易**：直观的交易界面，支持快速下单
3. **个人信息管理**：支持修改用户名等个人信息
4. **多标签界面**：清晰的标签页布局，方便切换不同功能
5. **响应式设计**：适配不同屏幕尺寸
6. **数据可视化**：使用进度条和颜色编码直观显示数据
7. **实时更新**：数据支持手动刷新，登录后自动加载用户数据
8. **用户友好**：清晰的标签页分类，直观的数据展示，统一的货币格式

## 技术实现

### 数据模型
- `QuoteData`：实时行情数据结构
- `KLineData`：K线数据结构
- `CompanyInfo`：公司信息数据结构
- `IndexData`：指数数据结构
- `SectorData`：板块数据结构
- `UserInfo`：用户基本信息
- `AccountInfo`：账户资产信息
- `CashFlowRecord`：资金流水记录
- `TradingStats`：交易统计信息

### 核心组件
- `DataService`：数据服务类，负责数据生成和管理
- `QuoteWidget`：实时行情显示组件
- `KLineWidget`：K线图绘制组件
- `CompanyWidget`：公司信息显示组件
- `UserService`：用户数据管理服务
- `UserInfoWidget`：个人中心界面组件
- `EditUserInfoDialog`：用户信息编辑对话框

### 模拟数据
当前版本使用模拟数据，包括：
- 随机生成的股票价格和成交量
- 模拟的K线数据
- 虚拟的公司财务信息
- 模拟的市场指数和板块数据

## 注意事项

- 本应用为模拟交易系统，不涉及真实资金
- 行情数据可能存在延迟，请以实际市场数据为准
- 请妥善保管您的账户信息

## 开发说明

### 添加新功能

1. 在 `head/` 目录下创建新的头文件
2. 在 `source/` 目录下实现相应的源文件
3. 在 `ui/` 目录下设计界面（如果需要）
4. 在 `StockTraderClient.pro` 文件中添加新文件
5. 重新编译项目

### 调试技巧

- 使用 Qt Creator 的调试器进行断点调试
- 在代码中添加 `qDebug()` 语句输出调试信息
- 检查 `build/` 目录下的编译输出和错误信息
